# TreeCraft AI Documentation

Complete guide to using TreeCraft AI - the intelligent file structure analyzer

## Overview

### What is TreeCraft AI?

TreeCraft AI is an intelligent web application that transforms project structures into beautiful visualizations with AI-powered insights. It provides an intuitive interface for analyzing file structures from GitHub repositories, manual input, or file uploads, and generates professional tree visualizations in multiple formats.

### Key Benefits

- Visualize complex project structures in an intuitive way
- Get AI-powered insights and documentation suggestions
- Multiple input methods (GitHub, manual input, file upload)
- Export in various formats (ASCII, Markdown, JSON, XML)
- Completely free and open source

## Features

### Multiple Input Methods
Import from GitHub repositories, upload files, or manually enter project structures with flexible input options.

### AI-Powered Analysis
Get intelligent insights and documentation suggestions powered by advanced AI algorithms.

### Multiple Output Formats
Generate visualizations in ASCII tree, Markdown, JSON, and XML formats for various use cases.

### Interactive Interface
Step-by-step guidance with a modern, responsive design that works on all devices.

### Export Options
Copy to clipboard or download your visualizations for use in documentation and presentations.

### Statistics Tracking
View analytics on files, folders, depth, and file types to understand your project structure.

## How to Use

### Getting Started
Using TreeCraft AI is simple and intuitive. Follow these steps to create beautiful project visualizations:

### Step 1: Choose Your Source
Select from three input methods:

- **GitHub Repository**: Paste a public GitHub URL to automatically analyze its structure
- **Manual Input**: Type or paste your file structure using standard tree notation
- **File Upload**: Drag and drop or browse to upload .txt or .md files containing file structures

### Step 2: Generate & Customize
Click "Generate Tree" to create your visualization. Use the AI analysis button to get intelligent insights about your structure. Customize the output format using the format tabs.

### Step 3: AI Analysis & Export
View statistics about your project structure. Copy the output to clipboard or download it. Use the AI documentation feature for enhanced insights.
